硬件设施

在北京理工大学的大力支持下，依托虚拟现实/增强现实技术及应用国家工程实验室、教育部光电成像重点实验室和北京市混合现实与新型显示工程技术研究中心，实验室拥有光学设计与VR/AR系统研制与实验的先进硬件条件。

单点金刚石车床

可用于制造光学透镜、光学模芯、金属反射镜和精密机械零件，加工球面、非球面和自由曲面等光学表面。该机床具备良好的车削性能，可实现行业领先的表面粗糙度Ra<3nm,面型精度PV<0.1μm，编程分辨率可达到0.01nm（直线）/0.0000001°（回转）

超精密注塑机

使用超精密注塑机，可实现球面、非球面、自由曲面光学镜片的快速验证，质量稳定。为相关理论验证与样机研制提供重要支撑。

光学面形检测设备

使用先进的三维立体测量仪，可用来测量非球面、自由曲面等光学面型，其测量角度可达75 °。探测角度在70 °情况下，测量精度可达到±0.15μm；测量角度在45°以内，精度可达0.05μm。

其它实验设施

实验室拥有光学平台、激光器、以及其它常用光学实验元器件等，方便研究人员开展相关实验验证工作。

软件平台

实验室拥有CODE V，LightTools，VirtualLab等业界最优秀的、最新光学设计与仿真软件，为开展理论算法、系统设计、仿真分析等研究提供强有力保障。

CODE V

实验室拥有业界最强大的最新成像系统设计软件CODE V（目前版本 CODE V 11.4）。实验室教师长年参与在全国范围内的CODE V软件培训教学工作，掌握顶级的设计经验与软件使用方法技巧。

LightTools

实验室拥有先进的照明设计软件LightTools，软件支持虚拟原型设计、仿真、优化和真实感图像渲染等照明设计应用,以及各种光束整形分析、杂光分析等（目前版本 LightTools 9.0）

VirtualLab

实验室拥有用于现代光学建模、分析和设计的统一化光学建模平台VirtualLab Fusion。建模范围包括了从几何光学到物理光学及其组合的简单或复杂的光学系统。

1stOpt

实验室拥有数值优化分析计算软件1stOpt，可用于复杂优化问题、非线性拟合、非线性方程组、参数估算反演、方程求解、微分方程求解、微分方程拟合、非线性规划、混合整数规划等。学校同时提供最新正版MATLAB软件。
